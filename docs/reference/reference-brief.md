# Aurelia Gems Reference Brief

Aurelia Gems is a premium jewellery and gemstone maison website.

Use the uploaded video and screenshots as implementation guidance for layout, spacing, visual rhythm, responsive behavior, and section order.

Do not copy any real luxury brand assets, logos, product names, exact text, trademarks, or copyrighted imagery into the final website.

## Homepage section order

The homepage must follow this exact order in the next phase:

1. Full-screen cinematic hero video
2. Full-width craftsmanship story section
3. Heritage / inspiration split section
4. Dark gemstone campaign section
5. Product and model split section
6. Boutique / appointment banner
7. Enchanting gifts campaign section
8. Our creations category grid
9. Journal / newsroom section
10. Services, newsletter, and footer

## Navigation reference

The website should use a luxury maison-style header:

* Fixed top header
* Centered “Aurelia Gems” wordmark
* Minimal menu trigger on the left
* Small utility icons/links on the right
* Transparent over the hero
* Solid ivory background after scroll
* Left-side menu drawer
* Dimmed page overlay
* Smooth premium animation
* Full-screen drawer on mobile
* Submenu support
* Keyboard accessible

## Design principles

The final website should feel:

* Premium
* Editorial
* Quiet luxury
* Gemstone-focused
* Spacious
* Manually art-directed
* Fully responsive
* Similar in quality and rhythm to a luxury maison website

Avoid:

* SaaS-style cards
* Generic AI-looking sections
* Cheap ecommerce design
* Sale badges
* Add-to-cart buttons
* Loud gradients
* Overused shadows
* Glassmorphism
* Exact copying of any real brand
