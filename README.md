# Aurelia Gems

Phase 1 foundation for a premium jewellery and gemstone maison website.

## Getting started

Install dependencies and start the Turbopack development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Reference Assets

Production assets used by the website are stored in `public/`. Design reference screenshots are stored in `docs/reference/` and are guidance only; they must not be rendered in the final website. The homepage implementation will happen in Phase 3.

## Quality checks

```bash
pnpm lint
pnpm build
```

The landing page, products, authentication, checkout, CMS, and backend are intentionally out of scope for Phase 1.
